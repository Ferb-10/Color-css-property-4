# Color-css-property-4

## Overview
This project is a simple web page designed to help beginners learn basic Spanish color vocabulary.
Each color is displayed with its Spanish name, styled text color, and a corresponding image.
The project was created to practice using HTML with external CSS for styling.

![image alt](https://github.com/Ferb-10/Color-css-property-4/blob/e6a455091fe0b4ebe1e74f840eac671fdc6aed88/assets/images/Color.png)

## Features
- Displays basic Spanish color vocabulary
- Each color name is styled to match its actual color
- Uses images to visually reinforce learning
- External CSS file for styling

## Technologies Used
- HTML5
- CSS

## What I Learned
- How to link an external CSS file to an HTML document
- Using id selectors to apply specific styles
- Using class selectors for shared styling
- Styling text color with CSS
- Controlling image size using CSS

## 概要
このプロジェクトは、スペイン語の基本的な「色」の単語を学ぶためのシンプルなWebページです。
それぞれの色は、スペイン語表記・対応する文字色・画像と一緒に表示され、視覚的に学べる構成になっています。
HTMLと外部CSSを使ったスタイリングの練習を目的として作成しました。

## 機能
-スペイン語の色の単語を表示
- 単語ごとに実際の色で文字を装飾
- 理解を助ける画像の表示
- 外部CSSファイルによるスタイル管理

## 使用技術
- HTML5
- CSS

## 学んだこと
- HTMLとCSSを連携させる方法
- idセレクタを使った個別スタイリング
- classセレクタによる共通スタイルの管理
- CSSで文字色を変更する方法
- CSSで画像サイズを調整する方法
